# WebAgente

Frontend web publico separado de `FrontAgente`.

## Inicio rapido

```bash
npm install
npm run dev
```

## Deploy en GitHub Pages

1. Crear `WebAgente/.env` (opcional) con:
```bash
VITE_GH_PAGES_BASE=/WebAgente/
```
2. Publicar:
```bash
npm run deploy
```

URL esperada:

`https://monraspgit.github.io/WebAgente/`
